# Tulipanes Moreno

Proyecto web informativo sobre los tulipanes, desarrollado como práctica académica de desarrollo web.

## Descripción

Este sitio web presenta información básica sobre los tulipanes, incluyendo su historia, significado y cuidados principales.  
El proyecto fue construido usando HTML y CSS, aplicando estructura semántica, metodología BEM, Flexbox, Grid y propiedades de posicionamiento.

## Tecnologías utilizadas

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Metodología BEM
