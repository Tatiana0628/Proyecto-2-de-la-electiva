Proyecto académico

¿Quiénes somos?

Este proyecto fue creado como parte de una práctica académica de desarrollo web, con el objetivo de aplicar correctamente etiquetas semánticas, estilos externos y una organización visual clara.

La temática de los tulipanes permite trabajar un sitio con contenido visual, secciones informativas y una presentación agradable para el usuario.

Nuestro objetivo

Presentar información de forma ordenada, con una estructura moderna y fácil de entender, demostrando el uso de HTML5, CSS, Flexbox, Grid y buenas prácticas de maquetación.
